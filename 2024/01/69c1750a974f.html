数组

  1. 声明数组：double[] hens={3.5,5.9,6.7,3.4,2.1,5.5};

//遍历数组的值

数组赋值机制

基本数据类型的赋值：值拷贝

数组默认情况下是引用传递，赋的是地址：地址拷贝，引用拷贝

int[] arr1={1,2,3}
int[] arr2=arr1;
arr2[0]=10;
//arr1={10,2,3}

修改arr2的值会导致arr1的值改变

数组拷贝

独立

int[] arr1={10};
int[] arr2=new int[arr1.length];
for(int i=0;i<arr1.length;i++){
    arr2[i]=arr1[i];
}

数组翻转

int[] arr={1,2,3,4,5,6};
int temp=0;
int len=arr.length;
for(int i=0;i<len/2;i++){
    temp=arr[len-1-i];
    arr[len-1-i]=arr[i];
    arr[i]=temp;
}
int[] arr={1,2,3,4,5,6};
int[] arr2=new int[arr.length];
for(int i=arr.length-1,j=0;i>=0;i--,j++){
    arr2[j]=arr[i];
}
arr=arr2;//让arr指向arr2数据空间，此时arr原来的数据空间就没有变量引用，会被当做垃圾销毁

二维数组

定义

int[][] arr={{1,2,3},{4,5,6},{7,8,9}};

arr[i]表示二维数组的i+1个元素

arr[i].length得到对应的每一个一维数组的长度

arr.length是一维数组的个数，因为二维数组的每一个元素是一维数组

二位数组的引用

  1. int a[][]=new int[2][3]

  2. 先声明在定义：

    类型 数组名[][];

    数组名=new 类型[][];

int[][] arr=new int[3][];
for(int i=0;i<arr.length;i++){
    //给每一个一维数组开空间new
    //如果没有给一维数组开空间，那么arr[i]就是NULL
    arr[i]=new int[i+1];
    for(int j=0;j<arr[i].length;j++){
        arr[i][j]=i+1;
    }
}

  1. 静态初始化

    int[][] arr={{1,2,3},{4,5,6},{7,8,9}};

二维数组的使用细节

  1. 一维数组的声明方式有：int[] x或int x[]
  2. 二维数组的声明方式有：int[][] y 或int[] y[] 或int y[][]
  3. 二维数组是由多个一维数组的组成的，它的各个一维数组的长度可以不相同
