伪代码

1.算法名称

  • 过程 Procedure: Procedure <算法名>([<参数列表>])
  • 函数 Function Function <算法名>([<参数列表>])

2.指令序列

书写规则：

  • 用Begin作为开始，用End作为结束；
  • 或者用“{”作为开始，用“/}”作为结束
    Begin
	指令序列;
End
或者
{
	指令序列;
/}

3.输入输出

输入：Input
输出：Output 或Return

4.分支选择

//第一种
If<条件> Then
	{
		指令序列;
	/}
//第二种
If<条件> Then
	{
		指令序列1;
	/}
Else
	{
		指令序列2;
	/}

5.赋值

x:=x+1;
x<-x+1;

6.循环

//for循环
For 变量：=初值 To 终值
{
	指令序列;
/}
//while循环
While (条件) do
{
	指令序列;
/}

7.算法结束

End <算法名称>

补充

数组写法

A[j]指示数组A的第j个元素。符号“ …”用来指示数组中值的范围。
例如：A[1…j]表示含元素A[1], A[2], … , A[j]的子数组；
还有个写法是A[0:n]表示数组下标从0开始一直到n
二维数组也是：A[0:m,0:n]

procedure Bubble(n:integer);
  var temp,i,j:integer;
  change:boolean;
  begin
     for i:=1 to n-1 do
     begin
       change:=false;
       for j:=n-1 downto i do
         if a[j]>a[j+1] then
         begin
           change:=true;
           temp:=a[j]; a[j]:=a[j+1]; a[j+1]:=temp;
         end;
         if not(change) then exit;
     end;
  end;
procedure insertsort(n:integer);
  var i,j:integer;
  begin
    for i:=2 to n do
    begin
      a[0]:=a[i];
      j:=i-1;
      while a[j]>a[0] do  
      begin
        a[j+1]:=a[j];
        j:=j-1;
      end;
      a[j+1]:=a[0];
    end;
  end;
