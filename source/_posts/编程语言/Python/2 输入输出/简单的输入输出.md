## 输入
```python 
s=input("提示字符串")
```
input函数以文本或字符串形式返回
可以用eval函数来求值并转换为一个数值


## 输出
print（）函数
参数可以是数字、字符串（要加引号）、表达式（会运算）

```python
#打印在文件中
fp=open('D:/text.txt','a+')
print('hello',file=fp)
fp.close()
```

不进行换行输出
```python
print ('hello','python','123')
```
